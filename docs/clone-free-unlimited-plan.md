# Бесплатный режим Клона без лимита сообщений

Цель ветки: сохранить три ответа до Telegram, затем разрешить свободное общение в бесплатном базовом алгоритме без продуктового лимита сообщений. Технические rate limits и защита от автоматизации сохраняются. Платный режим продаёт глубину полной карты, Паспорт клона, полный образ и Сонастройку, а не право отправлять сообщения.
