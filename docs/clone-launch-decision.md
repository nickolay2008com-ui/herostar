# Звёздный клон — решение по бесплатному режиму

## Стартовый оффер

- создание Звёздного клона бесплатно;
- первые 3 ответа доступны без регистрации;
- затем Telegram сохраняет клона, историю и доступ на любом устройстве;
- после входа бесплатный базовый алгоритм работает без продуктового лимита сообщений;
- технические rate limits защищают сервис от автоматизации и циклических запросов;
- платный «День со Звёздным клоном» продаёт глубину полной карты, связи аспектов, полный образ, карту и Паспорт, а не право продолжать переписку;
- Сонастройка остаётся отдельной 30-дневной программой.

## Главная рекламная цель

`clone_first_answer`

До накопления устойчивой статистики кампания запускается на контролируемые клики, а не на автоматическую оптимизацию по оплатам.
